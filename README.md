## AI-Powered Job Portal Platform

Full-Stack Job Board with AI-Powered Resume Matching. Built with Next.js 14, Drizzle ORM, and TypeScript. Features include role-based auth, automated candidate scoring, resume summarization, AI search (Prompt-based job search), and AI cron jobs.

### Getting Started

Install dependencies:

npm install
# or
yarn
# or
pnpm install
# or
bun installRun the development server:

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun devThen open `http://localhost:3000` in your browser.

### Scripts

- **dev**: Run the development server.
- **build**: Create an optimized production build.
- **start**: Start the production server.
- **lint**: Run linting.

### Deployment

You can deploy Axis to any platform that supports Next.js (e.g. Vercel). For details, see the Next.js deployment docs: `https://nextjs.org/docs/app/building-your-application/deploying`.
